Loop 3/4: l'agente con tool

Stesso loop conversazionale, con un sub-ciclo tool dentro al turno del modello

Il loop esterno è identico a prima. Cambia solo il nodo Modello: invece di rispondere subito, può emettere una tool call, attendere il risultato, e re-iterare N volte prima di chiudere il turno. Il numero di cicli interni non è predeterminato — lo decide il modello a runtime.

Limiti obbligatori in produzione: un tetto al numero di tool call per turno, un timeout per singola esecuzione, un budget di token per sessione. Senza, l'agente può entrare in cicli infiniti.
