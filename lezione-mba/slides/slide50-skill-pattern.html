Skill — il pattern tecnico

Una directory con documentazione, istruzioni, esempi, eventuali script. Un file principale (SKILL.md) spiega al modello cosa la skill offre e come usarla.

Esempio

Skill "prodotti e pricing" → directory con SKILL.md (istruzioni di navigazione), listino strutturato, regole di sconto, riferimenti a tool specifici.

Meccanica sottostante

In Claude Code, il meccanismo è il tool bash: il modello esplora la directory e legge i file. Altri harness usano meccanismi proprietari equivalenti.
