Skill: una directory con documentazione, istruzioni, esempi, eventuali script. SKILL.md spiega al modello cosa offre e come usarla.

Esempio

Skill "prodotti e pricing": SKILL.md, listino, regole di sconto, riferimenti a tool.

In Claude Code il meccanismo è il tool bash: il modello esplora la directory e legge i file.
